## Installation

To install Resume-Screening-project

1. **Clone the Repository**
First, clone the project repository from your version control system (e.g., GitHub). Open your terminal or command prompt and run:

```bash
git clone https://github.com/611noorsaeed/Resume-Screening-with-Machine-Learning-Job-Recommendations-Parsing-Categorization.git 
```

if you want to change this directory name 
```bash
ren Resume-Screening-with-Machine-Learning-Job-Recommendations-Parsing-Categorization
Resume_Screening_app
```

2. **Navigate to the Project Directory**
Change to the project directory:

```bash
cd Resume-Screening-with-Machine-Learning-Job-Recommendations-Parsing-Categorization
```
or
```bash
cd 
Resume_Screening_app
```

3. **Create a Virtual Environment**


For Windows:

```bash
python -m venv Resume_Screening_env
```
**Activate the Environment**
For powershell
```
.\Resume_Screening_env\Scripts\Activate.ps1
```

For cmd 
```
.\Resume_Screening_env\Scripts\Activate.bat
```
4. **Install Required Packages**


```bash
pip install -r requirements.txt
```

5. **Run the Application**
click on the play button on the top right

or 
```bush
flask run
```
